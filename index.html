Introduction

This is early alpha version! There are not all features yet implemented, not heavily tested with different devices and code might get large breaking changes until the first release.

Eliot is an open source system for managing containerized applications on top of the IoT device with an emphasis on usability, simplicity, security, and stability. Eliot gives simplified app delivery, isolation and additional security to traditional installations.

Docker and Kubernetes have inspired heavily and if you're familiar with those, you find really easy to get started with Eliot.
